# Pillars

## Core Gameplay Loops
### 3.1 Core Loop (1–3 минуты)
1) Запуск рейда → волны врагов.
2) Командование перемещением + стойки + ситуативные команды.
3) Победа/поражение → награда (валюта/ресурсы/ДНК/ингредиенты).
4) Улучшение питомца / бридинг / подготовка → следующий рейд.

### 3.2 Meta Loop (дни/недели)
- Формирование “идеального” питомца:
  - сбор ресурсов,
  - бридинг (наследование/комбинации),
  - подбор мутаций/кибер-модулей под модификаторы рейдов.
- Открытие новых арен/боссов/модификаторов.
- Коллекционирование косметики.

## Game Modes (overview)
### 4.1 Wave Arena (основной режим)
- Плоская арена, враги заходят с разных сторон.
- Длительность: 2–6 минут (настраиваемо).
- Цель: выживание/набор очков/достижение волны.
- Награды: базовая валюта + ресурсы под мутации/бридинг.

### 4.2 Boss Contracts (короткий бой с боссом)
- Длительность: 2–4 минуты.
- Босс с телеграфами и фазами; требует грамотного позиционирования и команд.
- Награды: ингредиенты для детерминированных апгрейдов/мутаций.

### 4.3 Modifiers System (контент-мультипликатор)
- Рейды получают модификаторы, которые меняют мету и билд-ценность:
  - усиление/ослабление типов урона,
  - ауры врагов (замедление/яд/щит),
  - лимиты лечения,
  - “элитные” волны,
  - повышенная агрессия/скорость мобов.
- Это основной способ добавлять контент без постоянного производства новых карт.

### 4.4 PvP (отложено)
- Приоритет: PvE.
- PvP при необходимости — асинхронное/ограниченное, без боевых расходников.

## Key Risks & Mitigations (overview)
1) Риск: игра воспринимается как idle/симуляция
   - Митигация: постоянное командование move + частые CP-решения.

2) Риск: P2W восприятие из-за зелий/ревивов
   - Митигация: зелья ситуативные, лимиты, отсутствие темпо-мультипликаторов, честный фритуплей доступ.

3) Риск: контент быстро кончается
   - Митигация: модификаторы рейдов, короткие режимы, быстрая ротация.

4) Риск: комплаенс вокруг рандома
   - Митигация: никаких покупок, влияющих на шанс; внутриигровая рандом-валюта не покупаемая.
