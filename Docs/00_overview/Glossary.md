# Glossary

- **Hub** — социальный place: станции, подготовка, витрина.
- **Raid Place** — отдельный place под рейды (приватные инстансы 1–4 игрока).
- **Reserved Server / Access Code** — механизм создания приватного сервера для рейда и телепорта пати.
- **Wave Arena** — основной режим: волны на плоской арене.
- **Boss Contract** — короткий бой с боссом (телеграфы/фазы).
- **Modifiers** — набор модификаторов рейда, меняющих мету/ценность билдов.
- **Pet Archetype** — роль питомца (Tank/DPS/Support).
- **Stance** — стойка поведения питомца (Attack/Def/Balance).
- **CP (Command Points)** — ресурс для команд в бою.
- **Mutation** — билд-направляющий эффект/прок (яд/щит/цепь/вампиризм и т.п.).
- **Cybernetics** — ветка модулей поведения/поддержки (в MVP отложено).
- **Breeding 2→1** — скрещивание двух питомцев в одного с наследованием/комбинациями.
- **Genome Capacity** — бюджет для боевых (combat) мутаций; ограничивает суммарную стоимость активных боевых эффектов.
- **Visual Channels** — каналы визуальных мутаций (Head/Torso/FullBody/Aura/Accessory/Color), по одному активному на канал.
- **Marionette** — боевая мутация: делает один случайный skill питомца тактическим (ручным) в рейде.
- **TacticalEligible** — флаг у skill, который разрешает его стать “тактическим” (ручным) через Marionette.
- **Fusion** — крафт/слияние мутаций по ограниченному списку рецептов.


- **Космокредиты** — базовая массовая валюта для покупок, сервисов и короткой петли прогресса.
- **Биомасса** — боевой органический материал для мутаций и лабораторных боевых операций.
- **Геном** — редкий генетический ресурс для breeding и роста Genome Capacity.
- **Хрома** — косметическая валюта для визуальных мутаций, аур и статусных улучшений.

- **Biome** — контентная тема рейда: визуал, семейства врагов, тип угроз, reward bias, пул модификаторов и минибоссов.
- **Tier** — ступень PvE-прогрессии рейда: длина, budget, элиты, минибоссы, trigger-плотность, базовое качество наград.
- **Difficulty** — scalar-слой рейда: статовые множители, число модификаторов, множитель награды, интенсивность trigger'ов.
- **Encounter Trigger** — сценарное событие вида “Condition → Action”, которое усиливает или меняет ритм рейда.
- **Budget-based Clear Percent** — расчёт % зачистки не по числу убитых мобов, а по budget-стоимости побеждённых врагов.
- **PacingProfile** — профиль темпа encounter-а (steady, swarm-heavy, elite-pressure и т.п.).
