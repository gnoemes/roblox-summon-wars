# Decisions

> Файл фиксирует уже описанные решения и ограничения из исходного Game.md, без добавления нового дизайна.

- Камера: бои в 2.5D top-down; хаб в 3rd person.  
- PvE — приоритет; PvP отложено, если будет — ограниченно и без боевых расходников.  
- В MVP: мутации есть; кибернетика — позже (или как косметика).  
- Рандом допускается только в бридинге/внутриигровых механиках с непокупаемой валютой; покупка влияния на шанс запрещена.  
- Модель мира: Hub Place + отдельный Raid Place с приватными инстансами (reserved server).  
- Награды выдаёт сервер рейда (server authoritative).
