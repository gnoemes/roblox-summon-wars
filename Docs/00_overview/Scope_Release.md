# Scope — Post-MVP (P1/P2)

Ниже — **существующее** продолжение Roadmap после MVP (без добавления нового).

### P1 (после первой валидации метрик)
- 10–20 модификаторов рейдов
- Daily/Weekly + streak
- 2–3 босса
- Расширение косметики и витрины
- Low/High VFX

### P2
- Ветка кибернетики
- Социальные системы (кланы/совместные контракты)
- PvP (если нужно) — ограниченно и без расходников
