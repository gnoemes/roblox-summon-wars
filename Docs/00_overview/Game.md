# INCUBATOR ZERO (Roblox)

Мультиплеерный хаб в стилистике милого sci-fi био-лаба + инстанс-рейды. Игроки разводят и усиливают “фэнтези-существ”, добавляя биомутации и кибернетику, и отправляют их в короткие PvE-рейды на арене. Бой: вид сверху (2.5D), питомец автоатакует, игрок командует перемещением (hold-to-move) и использует командные решения (стойки/приказы/расходники), ограниченные ресурсом.

## Основной цикл
- Хаб → подготовка (станции) → инстанс-рейд → награда → апгрейды/бридинг → следующий рейд.
