# Replication & Security (Draft)

> Этот файл пока содержит только то, что уже явно сказано в Game.md. Детальную модель безопасности/валидаций добавим позже.

## Server authoritative принципы (из Game.md)
- Награды выдаются сервером рейда перед возвратом в хаб.
- Рейды запускаются в приватных инстансах (reserved server), изоляция полная.
- Основная модель — Hub Place + отдельный Raid Place; внутриигровой “инстансинг” в одном мире не используется как базовый подход.

## Комплаенс вокруг рандома (из Game.md)
- Нельзя продавать шанс/удачу/повышение вероятности мутаций.
- Рандомные исходы — только через бридинг или внутриигровые механики с непокупаемой валютой.
