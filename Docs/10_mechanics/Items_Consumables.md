# Items / Potions (Consumables)

### 8.1 Ограничение
- **2 зелья на бой** максимум.

### 8.2 Балансная цель
- Зелья ситуативные, не “налог на прохождение”.
- Дорогие по внутриигровой валюте, но реально доступные регулярно за счёт грамотной экономики.

### 8.3 Типы зелий (предпочтительно)
- маленький хил,
- снятие дебаффа,
- короткий щит,
- одноразовый “анти-смерть” (мягкий сейв).

## Боевой UI (контекст)
- Большие зоны нажатия, минимальный текст, иконки.
- Боевой UI минималистичный:
  - HP питомца
  - CP
  - стойка
  - 2 слота зелий
  - 1–2 командные кнопки
