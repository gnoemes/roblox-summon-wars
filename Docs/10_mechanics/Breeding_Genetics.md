# Breeding / Genetics

### 6.1 Бридинг
- Формат: 2 → 1.
- Наследование:
  - часть мутаций/трейтов переносится по правилам,
  - возможны комбинации (если совместимые).
- Прозрачность:
  - игрок должен понимать вероятность/правила (не “чёрный ящик”).

### 6.2 Рандом и ограничения
- Рандомные исходы (мутации/комбо) существуют только:
  - через бридинг,
  - через внутриигровые механики с непокупаемой валютой.
- Запрещённая зона:
  - любые покупки, влияющие на вероятность/рандом, напрямую или косвенно.

## Комплаенс-ограничения (из монетизации)
- продавать шанс/удачу/повышение вероятности мутаций,
- продавать внутриигровую валюту, если она используется для рандома/бридинга,
- продавать боевые “темпо-мультипликаторы” (например, CP regen x2) как основной товар.
