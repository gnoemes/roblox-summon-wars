# Modifiers System

- Рейды получают модификаторы, которые меняют мету и билд-ценность:
  - усиление/ослабление типов урона,
  - ауры врагов (замедление/яд/щит),
  - лимиты лечения,
  - “элитные” волны,
  - повышенная агрессия/скорость мобов.
- Это основной способ добавлять контент без постоянного производства новых карт.
