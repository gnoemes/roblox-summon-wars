# Pet System — Archetypes / Stats / Slots

Архетипы питомцев (MVP)
- 3 стартовых архетипа:
  - Tank (выживаемость/агро),
  - DPS (урон/проки),
  - Support (хил/баф/дебаф).

### 5.2 Stats (минимум)
- HP, Damage, Defense, AttackSpeed
- Доп. параметры (по мере роста):
  - Crit, Lifesteal, StatusPower, CooldownReduction.

## Slots
- 2 слота мутаций в MVP.
- Расширение слотов — через прогрессию и/или честные покупки удобства (не рандом).
