# Skill Icons

Recommended workflow:
1. Put the original icon file in this folder.
2. Upload it to Roblox and copy the resulting asset id.
3. Put that id into the relevant skill definition as `iconAssetId`.
4. Keep the original file here so it can be re-uploaded later if needed.

Preferred format:
- PNG

Recommended size:
- 256x256 or 512x512

Naming suggestion:
- `fighter_guardian_shield_bash.png`
